Présentation

Phasmophobia est un jeu vidéo où les joueurs doivent identifier le type de fantôme qui hante un lieu (Poltergeist, Fantôme, Démon, etc.). Pour ce faire, le joueur est doté de divers gadgets qui l'aident à déterminer la bonne entité en fonction des preuves. Chaque fantôme possède trois preuves spécifiques permettant de l'identifier.

Lors d'un projet de classe en NSI, il nous était demandé de réaliser un site web en utilisant des bases de données SQL, que nous devions mettre en ligne et exploiter dynamiquement via PHP. J'ai donc créé trois tables : une pour les fantômes, une autre pour les objets et une dernière pour les preuves. Ce projet m'a permis d'apprendre les bases du PHP et des bases de données SQL, ainsi que l'utilisation d'AlwaysData, un service permettant d'héberger ces tables.

Le code

Voici, par exemple, un extrait de code qui permet de récupérer les fantômes associés à certains types de preuves :

$bdd = new PDO('mysql:host=sql109.infinityfree.com;dbname=if0_36117964_Phasmophobia', 'if0_36117964', '3B5cxq55rcbfYeQ');

// Récupérer les preuves
$query = 'SELECT ID AS evidenceID, Name AS evidenceName, Description AS evidenceDescription FROM Evidence';
$response = $bdd->query($query);
$evidenceData = $response->fetchAll(PDO::FETCH_ASSOC);

// Récupérer les fantômes et leurs preuves
$query = 'SELECT G.ID AS ghostId, G.Name AS ghostName, GROUP_CONCAT(E.ID) AS evidenceIDs
FROM Ghost AS G
INNER JOIN Ghost_has_Evidence AS GHE ON GHE.Ghost_ID = G.ID
INNER JOIN Evidence AS E ON GHE.Evidence_ID = E.ID
GROUP BY G.ID
ORDER BY G.ID ASC';
$response = $bdd->query($query);
$ghostData = $response->fetchAll(PDO::FETCH_ASSOC);

Les pages

Le site comportait plusieurs pages :

La page d'accueil (index) permettait de naviguer sur l'ensemble du site et d'accéder aux différentes tables.

La page "Entités" affichait toutes les entités de la base de données.

En cliquant sur un bouton, on pouvait voir le fantôme sélectionné avec une courte description et ses preuves. Une petite image correspondant au fantôme était également affichée.

La page "Objets" listait tous les objets de la base de données, en montrant leur prix et leur description. Il était possible de modifier le tier des objets (trois niveaux : mauvais, normaux et bons) et de voir les changements en direct.

La page "Preuves" permettait de cocher/décocher des preuves, et un script JavaScript mettait à jour dynamiquement la liste des fantômes correspondants. Passer la souris sur une preuve affichait également une image de celle-ci.
