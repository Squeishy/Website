Presentation

Phasmophobia is a video game where players must identify the type of ghost haunting a location (Poltergeist, Ghost, Demon, etc.). To do this, the player is equipped with various gadgets that help determine the correct entity based on evidence. Each ghost has three specific pieces of evidence that help identify it.

During a class project in NSI, we were asked to create a website using SQL databases, which we had to put online and use dynamically via PHP. So I created three tables: one for ghosts, another for objects, and a last one for evidence. This project allowed me to learn the basics of PHP and SQL databases, as well as the use of AlwaysData, a service for hosting these tables.

The Code

Here is, for example, a code excerpt that allows retrieving ghosts associated with certain types of evidence:

$bdd = new PDO('mysql:host=sql109.infinityfree.com;dbname=if0_36117964_Phasmophobia', 'if0_36117964', '3B5cxq55rcbfYeQ');

// Récupérer les preuves
$query = 'SELECT ID AS evidenceID, Name AS evidenceName, Description AS evidenceDescription FROM Evidence';
$response = $bdd->query($query);
$evidenceData = $response->fetchAll(PDO::FETCH_ASSOC);

// Récupérer les fantômes et leurs preuves
$query = 'SELECT G.ID AS ghostId, G.Name AS ghostName, GROUP_CONCAT(E.ID) AS evidenceIDs
FROM Ghost AS G
INNER JOIN Ghost_has_Evidence AS GHE ON GHE.Ghost_ID = G.ID
INNER JOIN Evidence AS E ON GHE.Evidence_ID = E.ID
GROUP BY G.ID
ORDER BY G.ID ASC';
$response = $bdd->query($query);
$ghostData = $response->fetchAll(PDO::FETCH_ASSOC);

The Pages

The site had several pages:

The home page (index) allowed navigation throughout the site and access to different tables.

The "Entities" page displayed all entities from the database.

By clicking a button, you could see the selected ghost with a short description and its evidence. A small image corresponding to the ghost was also displayed.

The "Objects" page listed all objects from the database, showing their price and description. It was possible to modify the tier of objects (three levels: bad, normal, and good) and see the changes in real-time.

The "Evidence" page allowed you to check/uncheck evidence, and a JavaScript script would dynamically update the list of corresponding ghosts. Hovering over evidence would also display an image of it.
