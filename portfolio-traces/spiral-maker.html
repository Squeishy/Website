  • Context: Personal Project

What would launch me into the most beautiful years of my life...

When I was in tenth grade, I was already interested in digital creation and wanted to learn to code. Spiral Maker was one of my first programming adventures that I completed, and it taught me several basics. Of course, the site's code itself is horrible, since it was my first website, but it was interesting to create.

Coded in two days, Spiral Maker serves to create spirals (if you're bilingual, this shouldn't surprise you) by modifying and playing with parameters to generate all kinds of spirals: round ones, big ones, spiky ones, beautiful ones... Whatever! Once your work is finished, you can save the image and use it as an illustration, but you could also go back and forward (undo redo), clear the canvas, etc...

This experience also taught me to use GitHub for the first time, a tool that would save my life in the years to come. I also started to understand CSS and HTML, but especially JavaScript, which dynamically generates spirals based on user choices.
